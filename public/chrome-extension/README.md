# Panela Ads Library Filter - Chrome Extension

Meta Ads Library sayfalarında sadece "Shop Now" ve "Şimdi alışveriş yap" butonları olan reklamları gösteren Chrome eklentisi.

## Özellikler

- ✅ Sadece "Shop Now" ve "Şimdi alışveriş yap" butonları olan reklamları gösterir
- ✅ Diğer reklamları otomatik olarak filtreler
- ✅ Filtrelenmiş sonuçları kompakt ve şık grid'de gösterir
- ✅ Yeni reklamlar yüklendiğinde otomatik filtreleme
- ✅ Performanslı ve hafif

## Kurulum

1. Bu klasörü zip olarak sıkıştırın
2. Chrome'da `chrome://extensions/` adresine gidin
3. Sağ üstte "Geliştirici modu"nu açın
4. "Paketlenmemiş uzantı yükle" butonuna tıklayın
5. Bu klasörü seçin
6. Extension aktif olacak!

## Kullanım

Extension yüklendikten sonra Meta Ads Library sayfalarında otomatik olarak çalışır. Herhangi bir ayar yapmanıza gerek yok.

## Teknik Detaylar

- Manifest V3 uyumlu
- Content Script ile sayfa filtreleme
- MutationObserver ile dinamik içerik takibi
- Performans optimizasyonu ile debounce/throttle

## Güncelleme

Yeni versiyonları indirmek için Ayarlar sayfasından tekrar indirin ve Chrome'da extension'ı güncelleyin.
